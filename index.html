What is SQL ?

Structured query language (SQL) is a programming language for storing and processing information in a relational database. A relational database stores information in tabular form, with rows and columns representing different data attributes and the various relationships between the data values.

Linguagem de consulta estruturada (SQL) é uma linguagem de programação para armazenar e processar informações em um banco de dados relacional. Um banco de dados relacional armazena informações em formato tabular, com linhas e colunas representando diferentes atributos de dados e os vários relacionamentos entre os valores de dados.

Why is SQL important?

Structured query language (SQL) is a popular query language that is frequently used in all types of applications. Data analysts and developers learn and use SQL because it integrates well with different programming languages. For example, they can embed SQL queries with the Java programming language to build high-performing data processing applications with major SQL database systems such as Oracle or MS SQL Server. SQL is also fairly easy to learn as it uses common English keywords in its statements.

A linguagem de consulta estruturada (SQL) é uma linguagem de consulta popular frequentemente usada em todos os tipos de formulários. Analistas e desenvolvedores de dados aprendem e usam SQL porque ele se integra bem com diferentes linguagens de programação. Por exemplo, eles podem incorporar consultas SQL com a linguagem de programação Java para construir aplicativos de processamento de dados de alto desempenho com os principais sistemas de banco de dados SQL, como Oracle ou MS SQL Servidor. SQL também é bastante fácil de aprender, pois usa palavras-chave comuns em inglês em suas instruções.

What are the components of a SQL system?

Relational database management systems use structured query language (SQL) to store and manage data. The system stores multiple database tables that relate to each other. MS SQL Server, MySQL, or MS Access are examples of relational database management systems. The following are the components of such a system.

Os sistemas de gerenciamento de banco de dados relacional usam linguagem de consulta estruturada (SQL) para armazenar e gerenciar dados. O sistema armazena várias tabelas de banco de dados relacionadas entre si. MS SQL Server, MySQL ou MS Access são exemplos de sistemas de gerenciamento de banco de dados relacional. A seguir estão os componentes de tal sistema.

  1. SQL table
  2. SQL statements
  3. Stored procedure
  4. Tabela SQL
  5. Instruções SQL
  6. Procedimento armazenado
